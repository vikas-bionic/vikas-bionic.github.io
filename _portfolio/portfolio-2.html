" collection: portfolio --- Cellulosomes are multi-enzyme complexes highly efficient at degrading crystalline cellulose. This research explores the intricate network of protein-protein interactions that govern how these complexes self-assemble. ### Key Approaches: * **Affinity Chromatography:** Purifying multi-domain recombinant proteins using Ni-NTA and size-exclusion chromatography. * **Binding Kinetics:** Quantifying the thermodynamic binding affinities between diverse dockerin variants and cohesin domains using **Microscale Thermophoresis (MST)** and UV-Vis/Fluorescence spectroscopy. * **Data Analysis:** Utilizing MATLAB and Python script pipelines to analyze binding curves and map interaction preferences. *Note: This work has resulted in a co-authored structural biology manuscript currently out for peer-review submission.* This is an item in your portfolio.
